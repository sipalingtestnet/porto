## Intruction

### install lz4
```
sudo apt install lz4 -y
```
## Stop Acre and Backup priv_validator_state.json
```
sudo systemctl stop acred
cp $HOME/.acred/data/priv_validator_state.json $HOME/.planqd/priv_validator_state.json.backup
rm -rf $HOME/.acred/data
```

### download data

```
curl -L https://snapshot.arable.spt-node.my.id/acre/acre-snapshot-20230207.tar.lz4 | lz4 -dc - | tar -xf - -C $HOME/.acred
mv $HOME/.acred/priv_validator_state.json.backup $HOME/.acred/data/priv_validator_state.json
```

### start node

```
sudo systemctl restart acred
sudo journalctl -fu acred -o cat
```

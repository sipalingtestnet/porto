## Intruction

### install lz4
```
sudo apt install lz4 -y
```
## Stop Acre and Backup priv_validator_state.json
```
sudo systemctl stop aplanq
cp $HOME/.aplenq/data/priv_validator_state.json $HOME/.planqd/priv_validator_state.json.backup
rm -rf $HOME/.planqd/data
```

### download data

```
curl -L https://snapshot.planq.spt-node.my.id/planq/acre-snapshot-20230207.tar.lz4 | lz4 -dc - | tar -xf - -C $HOME/.planqd
mv $HOME/.planqd/priv_validator_state.json.backup $HOME/.planqd/data/priv_validator_state.json
```

### start node

```
sudo systemctl restart aplanq
sudo journalctl -fu aplanq -o cat
```

## instruction
You can use our address book. stop your node then copy the following command to get the addressbook
```
wget -O addrbook.json https://snapshot.nibiru.spt-node.xyz/nibiru/addrbook.json --inet4-only
mv addrbook.json ~/.nibid/config

```
